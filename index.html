XaviExchange

Después de aprender a usar Vue.js decidí crear una aplicación que pudiera ayudar a la comunidad en internet. Fue así como nació XaviExchange. En esta aplicación web podemos usar los datos de la API CoinCap para determinar el valor actual de las criptomonedas más populares. Puedes darle un seguimiento personal y observar gráficamente los cambios de las últimas semanas, así como la mejor opción de cambio.

Desarrollado con:

  • JavaScript
  • Tailwind CSS
  • Vue.js
